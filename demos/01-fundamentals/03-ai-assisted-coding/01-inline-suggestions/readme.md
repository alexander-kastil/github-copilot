# Inline Suggestions

Master real-time code completion by leveraging inline suggestions as you type. These hands-on exercises demonstrate how adding meaningful comments and clear prompts shapes Copilot's understanding of your intent, producing exact implementations tailored to your needs.

## JavaScript - Password Validator

Create [password-validator.js](password-validator.js) and execute the following prompts:

Prompt:

```
Generate a password validator that will check the following rules: min 6 characters, on uppercase letter or special character.
```

Follow-up Prompt:

```prompt
@workspace how can i run `#file:password-validator.js`
```

## TypeScript - Food Model

Create [food.model.ts](food.model.ts) and execute the following prompts:

Prompt:

```prompt
Create a food model with the following properties: name, description, price, and in-stock. Use the constructor to initialize the properties of the class
```

Prompt:

```prompt
Explain the different uses cases for classes and literal types for typescript projects like angular
```

Prompt:

```prompt
Convert the food model to a literal type
```
